기본 프로필

- A팀 시각디자인학과 11학번 김범선


주제1. 화장품 관심 정도

- 화장품은 딱히 뭐..사용하지 않는다.

- 랩시리즈의 스킨 로션과 케어존의 수분크림을 사용한다.

- 제품을 사용하게된 이유는 쓰다 보니 좋아서 사용하게 되었다.



주제2. 화장품 구매시 고려사항

- 기능성 화장품은 사용하지 않는다.

- 화장품을 구매할때는 매장방문을 통하여 구매한다.

- 하지만 혼자가면 직원의 피부에 잘맞는다 등의 이야기로 속는 느낌이 들어 어렵다.

- 써본 제품중에서 그나마 괜찮은 것에서 비슷한것으로 사용한다.


주제3. 성분에 대한 관심 정도

- 화장품을 구매할때 성분은 고려햐지 않고 오로지 피부타입에 맞춰 구매한다.

- 화장품 분석 어플을 이용해 본적 없다. 있는지도 몰랐다.

- 천연 성분이 들어간 제품 사용해 본적 없다.

- 천연성분을 강조한 광고중 기억에 남는 광고는 없다. TV를 안본다.

- 성분에 대한 관심이나 필요성은 없는것 같다. 뭐가 들어가는지 모르겠다.



주제4. 화장품 사용시 특이사항

- 성분을 알기쉽게 설명해주는 사이트가 있어도 이용해 볼 생각은 없다.

- 관심있는 부분이 아니라  실시간검색어 1위를 하지 않는 한 방문자체를 하지 않을 것 같다.

- 사이트가 동화버전이라고 해서 신뢰성이 떨어질 것 같지는 않으나 페이스북 등으로 접촉을 하게 되면 좋을 것 같다.





기본 프로필

-A팀 시각디자인학과 13학번 민소영



주제1. 화장품 관심 정도

- 현재 화장품을 사용하고 있다.

- 하루에 4가지 종류 미스트, 로션, 썬크림, 쿠션을 사용

- 현재 사용하는 화장품을 쓰는 이융은 그냥 무난하여 장점도 없고 트러블도 없다

- 최근에 세일이나 신제품 출시, 후기를 보았을때 화장품을 구매 해야 겠다고 생각 했다.

- 성분 위주로 검색한 적은 기초 제품을 고를 때만 신경쓴다.



주제2. 화장품 구매시 고려사항

- 화장품을 구매할때 유분기 없고 수분기 많은 제품으로 선택한다.

- 화장품을 구매할때 인터넷 또는 오프라인을 이용하며 그때그때 다르다

- 화장품을 선물할때는 받는 사람의 피부타입을 고려하여 선택한다. 예를들어 여드름 피부에는 여드름 제품

- 기능성 제품은 사용한 적이 없다.

- 화장품을 구매할때 사람들의 후기와 부작용이 있는지 없는지를 먼저 본다.

- 가격보다는 성분을 고려한다.



주제3. 성분에 대한 관심 정도

- 엄마가 사주시는 제품을 같이 이용하는데, 엄마가 천연화장품을 좋아하신다.

- 기초화장품은 사이트보다는 엄마의 추천으로 사용하며 귀찮아서 찾아서 쓰진 않는다.

- 화장품 성분에 대해 알아 본적이 있다. 성분을 알기위해 사이트를 이용한다.

- 이니스프리나 스킨푸드와 같은 천연성분을 강조한 광고를 보고 특별히 든 생각은 모델이 예쁘지만 굳이 그렇게 광고해봤자 거기서 거기일 것같다. 제품보다는 브랜드 이미지라고 생각한다.

- '화해'같은 화장품 성분 분석 어플을 사용해 본적이 있다. 화장품을 구매할때 마다 검색해본다.

- 그 어플은 후기가 많다는 것과 이모티콘을 사용하여 위험정도를 표시해 주는 것이 좋았다.

- 하지만, 내가 찾는 제품이 없을때 많아서 아쉬웠다.



주제4. 화장품 사용시 특이사항

- 화장품을 중간에 그만 사용한적은 있다. 빨갛게 알레르기

- 맞지 않았던 화장품의 성분을 봐두고 다음에 참고한다.

- 사이트를 만든다면 백설공주 보다는 '건성'과 같은 피부타입이 눈에 띄도록 하는 것이 좋다.

- 사이트에서 성분에 대해 알기쉽게 동화적으로 설명한다면 사용은 할 것 이지만 너무 길다면 읽지 않을 것 같다.

- 메뉴에 피부 타입별로 적어 놓으면 좋을 것 같다.

- 깔끔하고 캐릭터가 조금씩 있는 것이 좋겠다.

- 피부 타입별로 구분해 놓으면 본인 피부에 맞는것을 잘 찾을 수 있을 것 같다.

- 타입별로 구분되어 있다면 방문해 볼 의향 있다.
